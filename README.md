# SafeCheck

Repository clean-room per la definizione e prototipazione del progetto SafeCheck.

La baseline documentale 0.1.3 contiene:
- contratto MVP;
- struttura PDF;
- piano test pilota;
- policy clean-room/IP;
- modello costi;
- specifica spike PDF;
- template JSON demo per verbale di sopralluogo sicurezza sul lavoro.

Il repository non deve contenere documenti reali di clienti, PDF reali, screenshot, file Excel, dati personali, template aziendali riservati o codice derivato da altri progetti.
